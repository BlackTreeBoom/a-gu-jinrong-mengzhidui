# a股金融梦之队 Skills

这个仓库收集可直接给 Codex 类代理加载的金融研究 skill。每个 skill 都尽量保持两个特征：

- 可以落到实际分析动作，而不只是人物简介
- 明确适用场景、常见误区、风险边界

## 使用原则

1. 先选最贴近任务的问题框架，而不是先选你最喜欢的人物。
2. 遇到人设化表达时，优先抽取其“判断流程”。
3. 如果资料来自公开讨论页或搜索引擎摘要，默认存在语境缺失，分析时要注明不确定性。
4. 所有结论都应回到独立验证，不直接把公开大 V 观点当成事实。
